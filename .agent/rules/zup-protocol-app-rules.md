# Zup Protocol App

This is the Zup Protocol (https://zupprotocol.xyz) Web app. An application made to find, manage & deposit into DeFi liquidity pools easily. The goal of this app is to mainly allow easy acessibility to liquidity pools from the expert, to the newcomer.

## Priority

The priority of this application is: Easy to use application. No bad UX, no use of complex jargons, etc. your grandma should be able to use it, and understand what is happening.

## UX/UI

The UX/UI of this project is made to be incredible, we want to build the prettiest + best to use application of whole DeFi. We leverage everything needed to build this.

UI: Animations, Good Looking Colors, Great Designs, Motions, Good to look at, perfect combinations. This is what we should never forget, every screen, every button, every text should be made with love and care, even the smallest things should be taken with care and built like a mother did to his kid.

UX: No hard to use screens, always prioritize minimalism, simplicity, no use of complex jargons, and no complex screens, anyone should be able to understand what is happening, and what are the next steps. Include deep explanaion about things, links, articles etc.

## Web3 Packages

We should leverage all the best Web3 packages to build a seamless, easy to use application. It should look like "it just works" in web3.
