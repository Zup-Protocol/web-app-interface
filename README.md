# Zup Protocol - Web Interface

Premium "Web Native" migration of Zup Protocol.

## 🚀 Stack

- **Framework**: [React](https://react.dev/) + [TanStack Router](https://tanstack.com/router) (SPA)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **UI**: [shadcn/ui](https://ui.shadcn.com/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/) (Framer Motion)
- **Language**: TypeScript

## 🧞 Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Installs dependencies                       |
| `npm run dev`     | Starts local dev server at `localhost:5173` |
| `npm run build`   | Build the production site to `./dist/`      |
| `npm run preview` | Preview the build locally                   |
| `npm run test`    | Run unit tests                              |

## 📁 Project Structure

- `src/components/ui`: shadcn components
- `src/routes`: Type-safe file-based routing
- `src/hooks`: Custom React hooks
- `src/lib`: Utility functions and helpers
- `src/styles`: CSS and global styles
- `src/providers`: Application context providers
